# MarkFlow

GFM対応のシンプルなMarkdownビューアです。

## 機能

- **GFMサポート**: GitHub Flavored Markdownに完全対応
- **カスタムCSS**: 設定画面でスタイルをカスタマイズ可能
- **ファイル操作**: ドラッグ&ドロップまたはメニューからファイルを開く

## 使い方

### ファイルを開く

以下の3つの方法でMarkdownファイルを開くことができます：

1. **メニューから**: `ファイル` → `ファイルを開く...` (Cmd+O)
2. **ドラッグ&ドロップ**: Markdownファイルをウィンドウにドロップ

### CSSのカスタマイズ

`表示` → `設定...` (Cmd+,) で設定画面を開き、カスタムCSSを編集できます。
